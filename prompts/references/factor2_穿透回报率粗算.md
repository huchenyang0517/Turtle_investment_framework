## 步骤3：粗算穿透回报率

```
支付率 M = 近3年均值（从 data_pack §6 股息历史 + §3 归母净利润，同币种计算）
  过去3年支付率序列：[X1%, X2%, X3%]
  支付率均值 = [值]%（M）
  支付率锚定值 = 承诺支付率（若有承诺，来源 data_pack §6.1 分红承诺）；若无承诺则使用 M（近3年均值）
  ⚠️ 支付率全部取自年报同币种计算，禁止使用 yfinance payoutRatio 字段

**⚠️ 分红承诺优先规则**（若同时存在多种承诺，优先采用「每股不低于X元」）：

  **(1) 每股不低于 X 元**：承诺股息率 = X / 当前股价，穿透回报率 = 承诺股息率 × (1−Q%) + O/市值（股息部分固定）

  **(2) 不低于归母净利润 Y%**（无每股金额承诺时）：支付率锚定值 = Y%，穿透回报率 = (C × Y% + O) / 市值。**需考虑归母净利润变化**：
    - 基准：用当前 FY 的 C 计算
    - 敏感性：归母净利润下滑时股息率同步下滑，须做 C 情景分析（如 C×0.9、C×0.8 时的穿透回报率）
    - 保守估计：可取近3年 C 的最低值，或当前 C × 0.85~0.9 作为基准，并在报告中说明

回购 O = 注销型年均金额（过去3年，从 data_pack §15 读取）= [值] 百万元

税率 Q（按持股渠道查表）：

| 上市地 | 持股渠道 | 适用税率 Q | 到手比例 | 说明 |
|-------|---------|-----------|---------|------|
| 港股H股 | 直接持有 | 28% | 72% | 企业所得税10% + 股息税20% |
| 港股H股 | 港股通持有 | 20% | 80% | 内地个人通过港股通免征企业所得税层 |
| 港股红筹/开曼 | 直接持有 | 20% | 80% | — |
| 港股红筹/开曼 | 港股通持有 | 20% | 80% | — |
| 港股涉美国来源收入 | 任一 | 加权计算 | — | 美国来源部分44%，其余按上方规则 |
| A股 | 持有 > 1年 | 0% | 100% | — |
| A股 | 持有1月-1年 | 10% | 90% | — |
| A股 | 持有 < 1月 | 20% | 80% | — |
| 美股 | 直接持有 | 30% | 70% | 美国预扣税（无税收协定） |
| 美股 | 直接持有（W-8BEN） | 10% | 90% | 中美税收协定优惠税率 |

注：
- 若用户未指定持股渠道，默认：港股用港股通税率，A股用长期持有税率，美股用W-8BEN税率
- 注销型回购不做税务折扣

综合股息税率 = [值]%（Q）

市值从 data_pack §1 读取。

**若 §6.1 有「每股不低于 X 元」承诺**：
  粗算穿透回报率 = (X / 当前股价) × (1 − Q%) + O / 当前市值

**若 §6.1 有「不低于归母净利润 Y%」承诺**：
  支付率锚定值 = Y%，粗算穿透回报率 = [C × Y% × (1−Q%) + O] / 当前市值
  须做归母净利润敏感性：列出 C×1.0、C×0.9、C×0.8 时的穿透回报率，评估业绩下滑对股息率的影响

**否则（无承诺，用历史支付率）**：
港股：
粗算穿透回报率 = [C × 支付率锚定值 × (1 − Q%) + O] / 当前市值

A股（长期持有）：
粗算穿透回报率 = [C × 支付率锚定值 + O] / 当前市值

粗算穿透回报率 = [值]%（R）
```

## 步骤4：否决门判断

```
Rf 从 data_pack §14 读取。
门槛 II 按因子4规则计算：
  港股 = max(5%, Rf + 3%)
  A股 = max(3.5%, Rf + 2%)
  美股 = max(5%, Rf_US + 3%)

粗算快速否决规则：

① 若 R < Rf（无风险利率）→ 直接否决，不进入因子3
   理由：粗算回报率连无风险都跑不赢，精算不可能逆转

② 若 Rf ≤ R < II × 0.5（不足门槛一半）→ 直接否决，不进入因子3
   理由：因子3的现金质量修正通常在 ±2pct 以内，无法弥补如此大的差距

③ 若 II × 0.5 ≤ R < II → 标注"因子2粗算边际不达标"，继续进入因子3
   理由：因子3的现金分析可能修正（如扣除项中成长性支出占比高）

④ 若 R ≥ II → 正常通过
```
